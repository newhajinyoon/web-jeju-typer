제주어 입력기 사용법

이 페이지에 있는 텍스트 상자에 제주어를 입력해보세요! 간편하게 제주어 입력기를 사용하려면 아래 스크립트를 포함하세요.

사용 방법

  1. 자음 + "ㅏㅏ" (ㅏ 두 번)을 입력할 시 아래아(ㆍ)가 입력됩니다.
  2. 자음 + "ㅑㅑ" (ㅑ 두 번)을 입력할 시 쌍아래아(ᆢ)가 입력됩니다.
  3. 종성 입력 차례일 때 "ㅁㅅ"을 입력하면 "ꥱ"를 종성으로 사용할 수 있습니다. (초성으로는 사용하지 못합니다.)

제주 특별 문자가 감지되었을 때 현재 입력 중인 칸에 제주어 입력 미리보기가 달리고, 띄어쓰기나 줄바꿈을 해야 자동으로 입력됩니다.

별다른 제약 없이 출처만 남기시면 사용이 가능합니다! (https://github.com/newhajinyoon/newhajinyoon).
